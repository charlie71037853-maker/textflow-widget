📥 다운로드가 시작되었습니다

처음 실행 시 OS의 보안 경고가 뜰 수 있습니다. 아래 단계대로 진행해주세요.

"Mac을 보호하기 위해 차단했습니다" 메시지가 뜨면

  1. 다운로드한 .pkg 파일을 더블클릭 → 차단 경고가 뜸
  2. 시스템 설정 열기 (Apple 메뉴 → 시스템 설정)
  3. 왼쪽 메뉴 "개인정보 보호 및 보안" 클릭 → 아래로 스크롤
  4. "보안" 섹션의 "그래도 열기" 버튼 클릭 → Touch ID 또는 암호 입력
  5. 다시 뜨는 경고창에서 "열기" → 설치 진행

※ macOS Sequoia(15.x) 이상은 우클릭→열기 방식이 더 이상 통하지 않아 이 단계가 필요합니다.
설치 후 Premiere Pro를 완전히 종료(⌘Q) → 재실행 → Window > Extensions > TextFlow Pro

"Windows의 PC 보호" 경고가 뜨면

  1. 경고창의 "추가 정보" 클릭
  2. 아래에 나타나는 "실행" 버튼 클릭
  3. SFX 안내창에서 "Extract" → 자동 설치

※ 코드 서명 없는 신규 배포 파일에 Windows가 일시적으로 표시하는 경고입니다 (악성 아님).
설치 후 Premiere Pro를 재시작하면 Window > Extensions > TextFlow Pro
